# Abdelrahman Sayed — Portfolio

Static one-page portfolio built with React and Vite.

## Scripts

- `npm run dev` — local development
- `npm run build` — production build
- `npm run preview` — preview production build

## Content

Edit `src/data/portfolio.js` for bio, experience, skills, and projects. Assets live under `src/assets/`.

## Contact form

Messages are sent to the inbox configured in `src/data/portfolio.js` via [FormSubmit](https://formsubmit.co). Activate the service once using the confirmation email sent on first submission.
